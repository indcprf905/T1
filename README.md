# ثيم نور (Noor Theme) - ثيم احترافي لمنصة سلة

![Noor Theme](assets/preview.jpg)

## 📋 نظرة عامة

ثيم **نور** هو ثيم احترافي متكامل مصمم خصيصاً لمنصة سلة، يهدف إلى تقديم تجربة تسوق إلكتروني متميزة للمتاجر الفاخرة والعطور والإكسسوارات. يتميز الثيم بتصميم عصري أنيق ووظائف متقدمة تساعد في زيادة المبيعات وتحسين تجربة المستخدم.

## ✨ المميزات الرئيسية

### 🎨 التصميم والهوية البصرية
- **ألوان متناسقة**: نظام ألوان احترافي (أساسي: #3A6B83، ثانوي: #F5F5F0، تميز: #E8B856)
- **خطوط عربية جميلة**: دعم خطوط Tajawal, Cairo, Inter
- **أوضاع ألوان متعددة**: فاتح، داكن، تلقائي
- **تصميم متجاوب**: يعمل بشكل مثالي على جميع الأجهزة

### 🛍️ ميزات التسوق
- **صفحة رئيسية متكاملة**: بانر ترحيبي + 3 بنرات تسويقية + منتجات مختارة
- **صفحة منتج متقدمة**: معرض صور + خيارات + تقييمات + مشاركة اجتماعية
- **صفحة فئات محسنة**: فلاتر متقدمة + ترتيب + عرض شبكي/قائمة
- **صفحة هبوط للعروض**: مع عداد تنازلي ومؤشرات ثقة

### 🚀 الأداء والتحسين
- **تحميل كسول للصور**: تحسين سرعة التحميل
- **محسن لمحركات البحث**: بيانات منظمة وmeta tags
- **ضغط الملفات**: CSS وJS محسنة
- **كاش ذكي**: تحسين الأداء

### 🛒 ميزات التجارة الإلكترونية
- **سلة تسوق ذكية**: حفظ محلي + إشعارات
- **قائمة المفضلة**: إدارة سهلة للمنتجات المفضلة
- **بحث متقدم**: مع اقتراحات تلقائية
- **مشاركة اجتماعية**: WhatsApp, Twitter, Facebook

### 🌙 ميزات متقدمة
- **الوضع الليلي**: تبديل تلقائي أو يدوي
- **تعدد اللغات**: عربي وإنجليزي
- **إشعارات ذكية**: تأكيدات الإجراءات
- **تحليلات مدمجة**: تتبع سلوك المستخدمين

## 🚀 التركيب والاستخدام

### المتطلبات الأساسية
- حساب على منصة سلة
- Salla CLI مثبت على جهازك
- Node.js (الإصدار 14 أو أحدث)

### خطوات التركيب

#### 1. تسجيل الدخول إلى Salla CLI
```bash
salla theme login
```

#### 2. تنزيل الثيم
```bash
git clone https://github.com/your-repo/noor-theme.git
cd noor-theme
```

#### 3. تهيئة المشروع
```bash
salla theme init
```

#### 4. رفع الثيم
```bash
salla theme upload
```

#### 5. معاينة الثيم
```bash
salla theme serve
```

### تخصيص الثيم

#### تعديل الألوان
قم بتعديل الألوان في ملف `twilight.json`:
```json
{
  "settings": {
    "colors": {
      "primary": {
        "default": "#3A6B83"
      },
      "secondary": {
        "default": "#F5F5F0"
      }
    }
  }
}
```

#### تخصيص المحتوى
- **النصوص**: عدّل ملفات الترجمة في مجلد `locales/`
- **الصور**: استبدل الصور في مجلد `assets/`
- **القوالب**: عدّل ملفات `.twig` في مجلد `templates/`

#### إضافة ميزات جديدة
```javascript
// في ملف assets/main.js
window.NoorTheme.cartManager.addItem(product, quantity);
window.NoorTheme.wishlistManager.toggleItem(product);
window.NoorTheme.notificationManager.show(message, type);
```
